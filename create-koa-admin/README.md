## 目录介绍
- web-admin：后台管理系统
  - admin-client：后台页面
  - admin-server：nodejs中间层，请求数据。
   

## 服务启动

- web-admin下：
  - 启动node服务以及前端： yarn dev:admin 
  - node服务器启动 yarn server
  - 前端启动：yarn admin-client
